Documentación Cuarentinistas

Las llamadas REST se realizan en:

  • Clientes: localhost:8080/cuarentinistas/rest/clientes
  • Cuentas: localhost:8080/cuarentinistas/rest/cuentas
  • Movimientos: localhost:8080/cuarentinistas/rest/movimientos
  • Movimientos by cbuSalida: localhost:8080/cuarentinistas/rest/movimientos/cbuSalida/
  • Movimientos by cbuDestino: localhost:8080/cuarentinistas/rest/movimientos/cbuDestino/
  • Inversiones: localhost:8080/cuarentinistas/rest/inversiones
  • Bonos: localhost:8080/cuarentinistas/rest/bonos
